# Nettineuvoja Admin UI

## Setup

Requirements: *Node*

```
$ sudo npm install -g grunt-cli bower
$ npm install
$ cp .env.example .env
```

Open `.env`, enter the missing variables and save the file

## Start Development Environment

```
$ npm start
```

## Create development build

```
$ npm run build
```

## Create distribution build

```
$ npm run dist
```
